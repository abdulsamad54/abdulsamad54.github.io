CNC Plotter (2017)

I made a Computer Numerical Control Plotter from CD drives. The stepper motor of the drive was the essential component that came into use. I used Arduino UNO with an ATmega328P microcontroller, I used the CNC code for Arduino from Github. For the stepper motor driver, I used L293D IC Motor shield from Adafruit. I used Processing IDE for uploading the G-code to the machine.

For more details visit:
CNC Plotter

Home automation (2018)

I developed a 4-channel home automation system which can control appliances over the internet from a mobile phone. This IOT project helped set timer in the app for the light or fan to switch off after the selected period of time. The project was based over BlynkAPI and Blynk Mobile App. After slight modifications I was able to use Google Voice Assistant and Alexa to control the power to the appliances. I used NodeMCU ESP8266 1.0 module to connect the receiver end to the internet and for the on/off mechanism I used a 4 channel Relay Module Board. NodeMCU was programmed using the ArduinoIDE and Blynk Library. This device is now employed in smart room.

For more details visit:
Home Automation
